# Lemonade

Lemonade is a pre-purchase reuse companion. Before someone buys something new, it helps them see whether something they already own can do the same job.

The product connects two surfaces:

- a WXT Browser Scout that pauses high-confidence shopping intent; and
- Lemonade Lane, the town dashboard where captured decisions continue through same-job matching, reuse or repair missions, Cooling, honest review, and persistent town activity.

Implementation is in progress. The current source of truth is the [production documentation](./docs/production/README.md), with the staged delivery plan in the [implementation roadmap](./docs/production/01-implementation-roadmap.md).

## Repository status

The repository baseline is established on `main`. Application packages are introduced from Roadmap Stage 2 onward.

## Requirements

- Node.js 22 or later
- pnpm 11.11.0 (managed with Corepack)

No environment variables are required for the P0 demo loop.
